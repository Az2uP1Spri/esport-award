À propos d'ESPORT AWARD

ESPORT AWARD est une cérémonie fictive qui célèbre l'excellence dans le monde de l'esport. Ce site est un prototype de plateforme de vote conçu pour démontrer un concept d'interface utilisateur futuriste et interactive.

Important : Les votes effectués sur ce site sont purement simulés. Aucune donnée n'est stockée en base de données, et aucun système d'authentification n'est implémenté. Il s'agit d'un prototype front-end uniquement.

Ce projet est un site statique 100% côté client, compatible avec GitHub Pages. Les animations et interactions sont gérées en JavaScript vanilla, et les données sont stockées localement dans le navigateur via localStorage.
